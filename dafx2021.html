Sound Examples

Below are a few sound examples comparing both scalings used in the article. The figures show the spectrogram of each sound. The SRER is given by SRER ≝ 20 ⁢ ⁡ log10 RMS ⁡ [ x ( n ) ] RMS ⁡ [ ε ( n ) ] , where x ( n ) represents the waveform of the original sound and ε ( n ) represents the waveform of the residual, defined as ε ( n ) ≝ x ( n ) -x̂ ( n ) , where x̂ ( n ) is the waveform of the sinusoidal model.
